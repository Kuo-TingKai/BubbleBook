## 🛠️ 生成指令

### 使用 AUTOMATIC1111 WebUI
```bash
# 安裝並啟動
git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git
cd stable-diffusion-webui
./webui.sh

# 在 WebUI 中使用上述提示詞
# 建議模型: DreamShaper, Deliberate, 或專為兒童插圖訓練的模型
```

### 使用 ComfyUI
```json
{
  "workflow": "children_book_illustration",
  "model": "dreamshaper_8.safetensors",
  "width": 1024,
  "height": 1024,
  "steps": 50,
  "cfg": 7.5,
  "sampler": "euler_a"
}
```

### 使用 InvokeAI
```bash
invokeai --model dreamshaper --width 1024 --height 1024 --steps 50 --cfg 7.5
```

## 🎨 後製建議

### 色彩調整
- 使用 GIMP 或 Photoshop 調整色彩飽和度
- 確保所有插圖使用一致的色彩方案
- 加強彩虹效果的透明度

### 文字整合
- 為每頁預留文字空間
- 使用圓潤字體與插圖風格匹配
- 確保文字與背景有足夠對比度

### 品質檢查
- 檢查每頁的視覺一致性
- 確保泡泡的透明效果正確
- 驗證兒童友好的色彩和構圖
